#### 1. Setting the Viewport Meta Tag

```
<meta name="viewport" content="width=device-width, initial-scale=1">
```

#### 2. Using Media Query (CSS)

```
@media (max-width: 800px) {}
```

### 3. Thinking about Break Points

```
PC: 1009px ~ maximum
Tablet: 801px ~ 1008px
Mobile: 0px ~ 800px
```
